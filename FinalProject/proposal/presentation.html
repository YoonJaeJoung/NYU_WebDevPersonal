The Vision

I plan to extend the Midterm Project into a highly interactive, completed web experience.

Expanding the timeline past Dada to cover crucial later movements, bringing the library up to the contemporary era.

Surrealism Pop Art Modern Architecture Abstract Expressionism Contemporary

Fulfilling the custom JavaScript requirement with engaging, playful elements:

3. Dynamic Filtering & Gallery

Building custom JS functionality for the Artworks/Gallery pages.

  • Instant, seamless UI updates
